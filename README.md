# 8-Queen Puzzle Using iterative repair algorithm 

## Usage

### Create a new `Board` instance and call `Board#solve` on the instance.
```
Board.new(board_length)

```
For example:
```
board = Board.new(8, required_answers: 2, filled_character: "Q")
board.solve
```
